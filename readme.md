# Descrição
Projeto para apresentação dos personagens de star wars

# Instalação
composer install 

# Executar o projeto
php -S 127.0.0.1:8090

# view 
http://127.0.0.1:8090/src/views/personagens.php

# endpoints

 * Peoples
 - people/ All peoples
 - people/name find by name people
 * Planets
 - planets/ All peoples
 - planets/1 find by id planet
